## 1. Вход в систему

Откройте браузер и перейдите по адресу:
```
https://api.nomads-area.com/admin/
```

Введите логин и пароль, которые были выданы при настройке системы. Если забыли пароль -- обратитесь к разработчику.

---

## 2. Главная страница

После входа вы видите список разделов. Основные разделы:

| Раздел | Что делать |

| **Туры** | Добавлять и редактировать туры |
| **Бронирования** | Обрабатывать заявки клиентов |

## 3. Настройки сайта

**Настройки сайта -> Настройки сайта**

Здесь хранятся контактные данные и общая информация компании, которая отображается на сайте.

### Что можно настроить:

**Контакты:**
- Телефон -- основной номер телефона
- WhatsApp -- номер для WhatsApp
- Email -- почта компании

**Социальные сети:**
- Ссылки на Instagram, Facebook, YouTube, TikTok, TripAdvisor

**О компании:**
- Текст "О нас" -- описание компании
- Видео о компании -- ссылка на YouTube-видео
- Лет опыта -- цифра, которая показывается на главной
- Количество туристов -- цифра статистики
- Количество маршрутов -- цифра статистики

**Политика конфиденциальности:**
- Ссылка на документ

> Настройки сайта существуют в единственном экземпляре. Не создавайте новую запись -- редактируйте существующую.

## 4. Туры

**Туры -> Туры**

### Создание нового тура

Нажмите **+ Добавить тур** в правом верхнем углу.

#### Основная информация

| Поле | Описание |
|------|----------|
| **Название тура** | Отображается на сайте |
| **Тип тура** | Групповой или Приватный |
| **Активен** | Включить, чтобы тур показывался на сайте |
| **Страна** | Выбрать из списка |
| **Город** | Основной город тура |
| **Категории** | Можно выбрать несколько (горный, культурный и т.д.) |

#### Параметры тура

| Поле | Описание |
|------|----------|
| **Сезон** | Круглый год / Тёплый / Зима |
| **Длительность (дней)** | Количество дней тура |
| **Сложность** | 1 -- Лёгкий, 2 -- Средний, 3 -- Сложный |
| **Максимум людей** | Для групповых туров |

#### Цена

| Поле | Описание |
|------|----------|
| **Цена** | Базовая цена за человека (для групповых) или минимальная (для приватных) |
| **Валюта** | USD или KGS |

#### Описание

| Поле | Описание |
|------|----------|
| **Описание** | Краткое описание тура |
| **Что включено** | Список через запятую или текстом |
| **Что не включено** | Список через запятую или текстом |

### Фотографии тура

В блоке **Фото тура** добавьте изображения:
- Нажмите **Добавить ещё одно Фото тура**
- Загрузите фото (JPG, PNG, рекомендуется 1920x1080)
- Первое фото будет обложкой в каталоге

### Программа по дням (маршрут)

В блоке **Маршрут по дням** добавьте описание каждого дня:

| Поле | Описание |
|------|----------|
| **Номер дня** | 1, 2, 3... |
| **Заголовок** | Краткое название дня |
| **Описание** | Подробное описание |
| **Высота** | Например: 3500м |
| **Пешая дистанция** | Например: 8 км |
| **Дистанция на машине** | Например: 120 км |
| **Проживание** | Юрта / Гостиница / Кемпинг |

### Даты заездов (только для групповых туров)

В блоке **Даты заезда** добавьте конкретные даты:

| Поле | Описание |
|------|----------|
| **Дата начала** | Дата старта тура |
| **Дата окончания** | Дата завершения |
| **Доступные места** | Количество свободных мест |

> Для приватных туров даты не добавляются -- клиент сам указывает желаемые даты при бронировании.

### Ценовые тиры (только для приватных туров)

В блоке **Цены приватных туров** укажите цену в зависимости от размера группы:

| Поле | Описание |
|------|----------|
| **Минимум людей** | Например: 1 |
| **Максимум людей** | Например: 3 (оставьте пустым для "и более") |
| **Цена за человека** | Например: 450 |

Пример заполнения:
```
1-2 чел -> 600$ за человека
3-5 чел -> 450$ за человека
6+  чел -> 350$ за человека
```

> Диапазоны не должны пересекаться. Система покажет ошибку если они накладываются.

### Точки маршрута (карта)

В блоке **Точки маршрута** добавьте координаты для отображения на карте:
- Найдите точку на Google Maps
- Скопируйте широту и долготу
- Добавьте название точки

### Сохранение

Нажмите **Сохранить** в правом нижнем углу. Тур появится на сайте если стоит галочка **Активен**.

## 7. Бронирования

**Бронирования -> Бронирования**

Здесь отображаются все заявки на туры.

### Статусы бронирования

| Статус | Описание |
|--------|----------|
| **Ожидает подтверждения** | Новая заявка, оплата ещё не поступила |
| **Подтверждён** | Оплата получена, место зарезервировано |
| **Отменён** | Клиент или менеджер отменил |
| **Отклонён** | Отклонено (нет мест, ошибка оплаты) |

> Отменить бронирование со статусом **Подтверждён** через интерфейс нельзя. Если клиент уже оплатил и требует отмены -- обратитесь к разработчику.

### Как обрабатывать бронирование

**Если оплата через FinikPay:** система подтвердит автоматически после получения оплаты.

**Если оплата вручную (наличные/перевод):**
1. Откройте бронирование
2. В разделе **Платежи** найдите связанный платёж
3. Откройте платёж -> нажмите действие **Подтвердить оплату вручную**
4. Статус бронирования изменится на **Подтверждён**, места спишутся

**Массовые действия:**
- Выберите несколько броней галочкой
- В меню "Действие" выберите **Подтвердить и зарезервировать места** или **Отменить выбранные брони**

### Поля бронирования

| Поле | Описание |
|------|----------|
| **Клиент** | Имя и контакт |
| **Тур** | Название тура |
| **Дата тура** | Для групповых туров |
| **Желаемые даты** | Для приватных туров |
| **Количество людей** | Взрослые + дети |
| **Цена за человека** | Рассчитана системой |
| **Итоговая цена** | Полная стоимость |
| **Предоплата** | 30% от итоговой суммы |
| **Статус** | Текущий статус |

Здесь отображаются все платёжные записи.

### Статусы платежа

| Статус | Описание |
|--------|----------|
| **Ожидает оплаты** | Ссылка на оплату создана, оплата не поступила |

Если клиент оплатил наличными или переводом:
1. Найдите нужный платёж
2. Выберите его галочкой
3. Действие -> **Подтвердить оплату вручную**

Это автоматически подтвердит бронирование и спишет места.

## 11. Квиз и лиды

**Квиз -> Заявки с квиза**

Лиды от пользователей, прошедших квиз на сайте.

- Изменить порядок вопросов

## 12. Отзывы

Собственной базы отзывов в системе нет. Отзывы на сайте отображаются через два внешних виджета.

### Виджеты

| Виджет | Тип | Настройка |
|--------|-----|-----------|
| **Google Reviews** | Elfsight (платный) | App ID в поле `elfsight_google_reviews_app_id` |
| **TripAdvisor** | Официальный виджет TripAdvisor (бесплатный) | Embed-код в поле `tripadvisor_widget_code` |

### Как подключить

**Google Reviews (Elfsight):**
1. Зарегистрируйтесь на elfsight.com
2. Создайте виджет Google Reviews
3. Скопируйте App ID
4. Вставьте в Django Admin: Настройки сайта -> Виджеты отзывов -> поле **Elfsight Google Reviews App ID**

**TripAdvisor (официальный виджет):**
1. Войдите в аккаунт на tripadvisor.com
2. Перейдите в раздел управления бизнесом -> виджеты
3. Скопируйте embed-код
4. Вставьте в Django Admin: Настройки сайта -> Виджеты отзывов -> поле **TripAdvisor Widget Code**

### Включить/выключить виджеты

В Django Admin: Настройки сайта -> переключатель **Виджеты отзывов включены**.

## Частые вопросы

**Тур создан, но не отображается на сайте?**
Проверьте что стоит галочка **Активен** у тура.

**Клиент говорит что не может забронировать -- нет мест?**
Зайдите в тур -> Даты заезда -> проверьте поле **Доступные места**.

**Бронирование не подтверждается автоматически?**
Проверьте статус платежа. Если оплата наличными -- подтвердите вручную.

**Не приходят уведомления в Telegram?**
Сообщите разработчику -- нужно проверить токен бота.

**Как изменить цену тура?**
Туры -> найдите тур -> измените поле **Цена** -> Сохранить. Изменение не затронет уже созданные бронирования.

**Клиент хочет отменить бронирование?**
Если бронирование ещё не оплачено -- Бронирования -> действие **Отменить выбранные брони**. Если уже оплачено -- обратитесь к разработчику. Возврат денег -- вручную.

**Не работает виджет отзывов?**
Проверьте что переключатель **Виджеты отзывов включены** активен в Настройках сайта. Если включён -- проверьте правильность App ID (Google) или embed-кода (TripAdvisor).
